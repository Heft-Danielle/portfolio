Applied Coding

The Applied Coding course is a project-based, student-centered coding curriculum designed to build real-world programming skills through engaging, hands-on applications. Developed as an award-winning thesis project and implemented at the district level, this course blends creativity, technology, and personalized learning strategies to make coding accessible and meaningful for all students.

  • Real-World Tech Integration: Students build projects using platforms and tools such as Micro:Bits, EarSketch (music-based Python coding), Arduinos, and LED components.
  • Student Ownership: Learners co-create personalized assessment rubrics based on SMART goals, encouraging autonomy, reflection, and growth.
  • Cross-Disciplinary Alignment: Aligned with Michigan Computer Science Standards (MI-CS), Next Generation Science Standards (NGSS), and Common Core Mathematics to ensure academic rigor and relevance.
  • Equity-Focused Design: The course scaffolds content for diverse learning needs, prioritizing inclusivity, differentiated instruction, and creative expression.

Applied Coding Overview

Unit 1: Basics & Review of Coding

Students demonstrate coding skills by composing music with Python, personalizing code, setting clear goals and deadlines, troubleshooting errors, and comparing syntax across JavaScript, Python, and C++ to deepen their understanding.

Unit 2: Sensors

Students build and customize fully functional prototypes by wiring LEDs, soldering components, troubleshooting circuits, writing and testing code, using real-time sensor data, and setting clear goals. Their work focuses on creating practical, user-friendly solutions.

Unit 3: Lights

Students create functional prototypes by assembling components like 3D-printed parts and LEDs, soldering, and troubleshooting electronics. They customize code using libraries such as FastLED, program unique lighting effects, write clean, bug-free code, and manage their projects with clear goals and timelines.

Unit 4: Capstone

Students complete the course by independently creating functioning prototypes that use sensor data as program input, program individually addressable LEDs with color effects, and incorporate new hardware or software extensions beyond prior lessons.
