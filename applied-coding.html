Applied Coding

Northview Public Schools

The Applied Coding course is a project-based, student-centered coding curriculum designed to build real-world programming skills through engaging, hands-on applications. Developed as an award-winning thesis project and implemented at the district level, this course blends creativity, technology, and personalized learning strategies to make coding accessible and meaningful for all students.

  • Real-World Tech Integration: Students build projects using platforms and tools such as Micro:Bits, EarSketch (music-based Python coding), Arduinos, and LED components.
  • Student Ownership: Learners co-create personalized assessment rubrics based on SMART goals, encouraging autonomy, reflection, and growth.
  • Cross-Disciplinary Alignment: Aligned with Michigan Computer Science Standards (MI-CS), Next Generation Science Standards (NGSS), and Common Core Mathematics to ensure academic rigor and relevance.
  • Equity-Focused Design: The course scaffolds content for diverse learning needs, prioritizing inclusivity, differentiated instruction, and creative expression.
  • Award Recognition: Recognized by Grand Valley State University with the Outstanding Masterâ€™s Thesis Award for innovation in coding curriculum design.

This curriculum reflects my commitment to building inclusive, future-ready learning experiences that merge technical skills with creativity, purpose, and student voice.
